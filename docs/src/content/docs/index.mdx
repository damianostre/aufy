<CardGrid stagger>
	<Card title="Built on top of Identity framework" icon="pencil">

        authentication flows.
	</Card>
	<Card title="Web API included" icon="add-document">
        Aufy comes with a set of pre-built API endpoints to handle authentication.
        Built with minimal API for maximum performance.
	</Card>
	<Card title="JWT Bearer authentication" icon="setting">
        Aufy supports JWT Bearer authentication with refresh tokens.
	</Card>
	<Card title="Email and password authentication" icon="open-book">

	<Card title="Easily customizable" icon="open-book">
